[Mocha](https://mochajs.org/) testing framework

[Chai](https://www.chaijs.com/) assertions

---

## Prerequisites

You'll need a recent [Node.js](https://nodejs.org/) version. Then download this project's dependencies with:

    npm install

## Developing

Run tests once

    npm run test

Run tests continuously

    npm run autotest

Code reformat

    npm run format
